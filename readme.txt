== Description ==

Your niche website may have lot of traffic, but you may not get a good sale from your site because your call to action buttons is not properly optimized.

I have developed this Crazy Call To Action BOx plugin for myself, after getting a good success, I have shared this for all affiliate marketers, who need to increase their conversion rate.

This simple plugin will allow you to add 4 types of call to action link.

1.simple text link
2.awesome button
3. Two types of Image box with button.[See screenshot for details]

During my initial test with a niche website, this plugin increased `20%` conversion rate.


= how to use =
Use your wordpress post editor and here you can see crazy call to action button named "CCTAB" [please see Screenshots] just use this button to add any shortcode you want. 

It is easy and funny :)

You can insert simple text link, awesome buttons link, two different types image box to your website.

 This plugin will increase your conversion rate up to 20%

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `CallToActionBox` directory to the `/wp-content/plugins/` directory 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use `CallToActionBox` Button at content editor
